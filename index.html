Out Of Core Module

Support Big Image Operation (larger than 10G or more) : store, save, read and show

Download Download View Out Of Core Module on GitHub

Information

Out Of Core Module is designed for easy operation on big image(even larger than 10G). You can store a very big image in a low memory computer, manipulate the pixels, save it into the disk, and even read it back later for observation. The Core Design of the module is to support big image operation even in low memory computer.
