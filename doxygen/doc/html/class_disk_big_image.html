详细描述

template<typename T>
class DiskBigImage< T >

Derived from DiskBigImageInterface, support for the reading and writing of the big image that was saved in the disk.

The big image saved in the disk comes from the BlockwiseImage class or HierarchicalImage class. This class is used to access the image data in the disk at a low cost of using memory usage. When access the image data, the last frequently used image data will be save in the memory, and the none frequently used image data will be swap out to the disk.

模板参数
TThe type of the image cell
