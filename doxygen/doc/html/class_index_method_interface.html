virtual IndexType IndexMethodInterface::get_row_result ( RowMajorIndexType  row_index) const
pure virtual

get the row result from just the row index (for using row-major like loop). 

Using this method for some kind of optimization when using row-major like loop 
for example:    
for(size_t row = 0; row < rows; ++row) { 
        IndexType row_result = get_row_result(row);
        for(size_t col = 0; col < cols; ++col) { 
                IndexType index = get_index_by_row_result(row_result, col); 
        } 
}
参数
row_indexthe index of row in the row-major format
返回
the row result

在 ZOrderIndex,ZOrderIndexIntuition , 以及 Block2DIndex 内被实现.
